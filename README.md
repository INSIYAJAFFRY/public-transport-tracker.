# Public Transport Tracker & Route Optimizer

## Problem
Commuters face difficulty in tracking public transport and choosing optimal routes.

## Solution
A web-based MVP that allows users to enter source and destination and receive optimized public transport route suggestions.

## Features
- Source & destination input
- Route suggestion logic
- Simple and user-friendly UI
